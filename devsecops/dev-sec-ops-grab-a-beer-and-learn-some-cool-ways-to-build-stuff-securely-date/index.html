DevSecOps! Grab a beer & learn some cool ways to build stuff securely

When: Wed 29 November | Location Ovo Amphitheatre:  | Time: 6 - 9pm

What is DevSecOps?? Like DevOps, but the DevSecOps approach looks to incorporate security principles.

This is our inaugural Meetup and we'll likely have folk coming from Bristol, Reading and even London for the evening.  

Two speakers will be joining - more details soon

Topics:

Container security: Probing how containers solve/ create security challenges (and ways to reduce the container attack surface)

Optimised  CI/CD security testing (rather than the monolithic security testing which doesn't scale in a CI/CD environment)
