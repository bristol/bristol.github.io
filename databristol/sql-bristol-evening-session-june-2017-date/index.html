SQL-Bristol Evening Session (June 2017)

Chris Webb

Introduction to Azure Analysis Services (Business Intelligence)

With the release of Azure Analysis Services we have, at long last, got a crucial piece of the Microsoft BI stack available in the cloud. Azure Analysis Services is a SaaS version of Analysis Services, and in this session you’ll learn about:

• What Azure Analysis Services is

• When you should use it

• Configuring Azure Analysis Services in the Azure portal 

• Developing and deploying Azure Analysis Services models 

• Building reports in Excel and Power BI using Azure Analysis Services 

• Connecting to on-premises data sources from Azure Analysis Services 

• Automation with PowerShell 

• Integration with other cloud BI services 

• Sizing and pricing 

• Monitoring

Prathy Kamasani

What's new in SQL Server Reporting Services (Business Intelligence)

After years of abandon, Microsoft exhibited great love towards SSRS with SQL Server 2016 release. But it doesn't stop there. There are some exciting features in the preview for the next release. 

In this session, Prathy will show top features of SSRS 2016 and vNext. Various ways of designing and publishing reports to SSRS web portal using SQL Server Data Tools (SSDT), Report Builder, SQL Server Mobile Report Publisher and Power BI (Technical Preview of Power BI reports in SQL Server Reporting Services)
