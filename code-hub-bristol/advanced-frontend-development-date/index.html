Advanced frontend development

This session is aimed at the mid-level front-end developers who want to improve their front-end skills and someone who is interested in the advertising front-end development. In this session, I will present following items:

• Interpolation
• Tween Engines
• Canvas effects
• Basic GLSL (Using THREE.JS and/or PIXI.JS)
• Optimization
• How to be a better front-end developer
