South West Ruby #24

Everyone is welcome at Bristol's monthly Ruby meetup. Just started learning Ruby? Thinking about starting to learn? Been writing Ruby for years? Please come along! Meet each other, eat delicious food, and listen to talks about Ruby. Because Ruby really is great: https://blog.teamtreehouse.com/why-ruby-is-awesome.

This month, Miles Woodroffe (Cookpad's CTO) will be giving a talk about Feature Toggles and how Cookpad use them to experiment with new features and improve the development process.

There's room for another talk, so if you have something you'd like to present - a demo, a lightning talk, even some code you'd like reviewed! - please get in touch.
