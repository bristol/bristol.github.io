Nodecopter - Commanding flying robots with Javascript

Andrew Nesbitt is going to talk about hacking with 

Nodecopters!

We'll learn how to operate a quadcopter using code rather than your phone, make it take off, fly around and flip with just a few lines of javascript.

Then we'll push it even futher and try out some of the open source projects to control them in weird and wonderful ways with face detection, QR code controllers, streaming video and the latest web browser technologies.
