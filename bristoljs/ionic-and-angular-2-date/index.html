Ionic & Angular 2

<a>Ionic - Tom Kelsey</a>

Ionic is a framework built on top of Cordova for developing mobile apps with HTML, CSS and Javascript. 

I’ll run through my own experiences with Ionic, reel off some pros/cons and try to give some guidance as to what apps/situations best lend themselves to a hybrid framework like Ionic.

If there’s time - I’ll squeeze in a bit about the Ionic ecosystem and try and cover what’s happening with the development of Ionic v2.

I doubt they’ll be much, if any, actual code (I’m not sure if that’ll be met with sighs of relief or sadness?) but perhaps Joseph will wet your appetite on that front with his Angular 2 talk.

<a>Angular 2 - Joseph Woodward</a>

You've probably heard of all the drama surrounding  the Angular 1 to Angular 2 upgrade. I'll take you through what's changed, and why, features and lessons learned whilst building a number of applications using Angular 2 and TypeScript.
