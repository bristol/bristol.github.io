(sub group) Bristol Creative Code (ALL LANGUAGES)

• What we'll do
This meetup is for anyone who is interested in using code creatively, physical computing and digital art. Including Unity, Open Frameworks, Processing, Pure Data etc.

The format of this first meetup will be:

Welcome & Meet everyone
Show & Tell: present a project if you wish
Work on a personal project / Look at Open Frameworks together As this is the first Meetup, we will have a discussion about how we might format the meetup so that it is helpful and fun for everybody,

In the future we may run:
Workshops
Hacknights

• What to bring
A laptop and and an open mind
