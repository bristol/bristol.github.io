Bristol Creative Code #3: Audio

This meetup is for anyone who is interested in using code creatively

The format will be
Work on an existing audio project in a language / environment of your choice with the aim to Jam together (or make random noises) towards the end of the session

New creative coders may like to use Gibber
https://gibber.cc/

Creative Clojurians may like to use Overtone
http://overtone.github.io/

There are many other options including:
Sonic Pi, Pure Data etc.

Its a good idea to have something working before you come to the session because there is not a lot of time otherwise.
