Python DBBUG social

Its been a while since DBBUG last met up. 

So to beat the Christmas rush, lets get together for a social meetup at the end of November. 

It would be good to see some of the old members of DBBUG - along with new faces.

To talk about Python development, Django, DevOps and the like.

To help advertise it to the wider local Python community we have added a meetup site to the traditional organisation tool of the DBBUG Google Group that has been going for the last 7 years at https://groups.google.com/forum/#!topic/dbug/
