Web Development: Two Ways (with Shawn Wildermuth)

Web Development: Two Ways 

In stealing from a common chef theme of preparing a food two ways, I’ll be in Bristol to talk about two different ways of doing web development. 

First up will be an introduction on Web API 2 and why it’s the best way to build REST APIs using the Microsoft Stack. We’ll delve into the why as well as the how using the newest features of ASP.NET Web API 2.1. 

In the second part, I’ll switch gears and introduce you to Node.js from the perspective of the .NET developer. I’ll show you why I think learning Node.js will make you a better ASP.NET developer. It’ll be fun!
