A man, the "Roslyn" compiler and an Azure VM. A peek into the .NET Compiler

Feature Talk: A man, the "Roslyn" compiler and an Azure VM. A peek into the .NET Compiler Platform by Martin Beeby

Project Roslyn now called The Microsoft .NET Compiler Platform is a complete renewal of the C# and Visual Basic compilers, exposing them as full fidelity APIs for everyone to use, and providing a great foundation for evolving the tool experience and the languages themselves.

In this hour long session Martin will show you what the new compiler looks like, how to use it and what it’s capable of.

About Martin Beeby

Martin Beeby works for Microsoft where he talks to developers about HTML 5, Windows 10 and the web. Martin has been developing since he was 16 and over the past 17  years has worked on projects with many major brands

Martin has written articles for, and been featured in, .NET Magazine, Creative Bloq, SitePoint, ZDNet, the Microsoft Developer Network, and many more.

Lightning Talk: Let them use Cake

Cake (C# Make) is a cross platform build automation system with a C# DSL (domain specific language). This talk will discuss the reasons for using Cake, describe the technology involved and include a live coding demo or two.
