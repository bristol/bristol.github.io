Pushing the Performance of .NET Applications (with Marc Gravell)

Performance is a feature. 

In this session, Marc (one of the developers behind Stack Exchange / Stack Overflow) discusses some approaches for measuring and improving the performance of your .NET applications, using some of the freely available tools that the Stack Exchange team developed in order to achieve very high scalability and performance. 

In the second half of the session, Marc will be looking in particular at redis, which Stack Exchange use extensively (in combination with SQL databases), looking ahead to redis "cluster", and discussing ways in which you can use tools like this to improve the performance of your applications.
