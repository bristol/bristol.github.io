Containerised ASP.NET Core apps on Kubernetes

Feature Talk: Containerised ASP.NET Core apps on Kubernetes

Kubernetes is an open-source container management platform that has been available to the Linux world for a while. With ASP.NET Core, it's now available to .NET developers too! In this talk, we'll take a look what Kubernetes is and how it can help you with deployment and running of containerised ASP.NET Core apps.

About Mete Atamel:

Mete Atamel is a Developer Advocate at Google, currently focused on helping developers with Google Cloud Platform. As a long-time Java and a recent C# developer, Mete likes to compare the two ecosystems.

Prior to Google, Mete worked at Microsoft, Skype, Adobe, EMC, and Nokia building apps and services on various web, mobile and cloud platforms
