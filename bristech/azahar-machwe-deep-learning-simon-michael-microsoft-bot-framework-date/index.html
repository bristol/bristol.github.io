Azahar Machwe - Deep Learning / Simon Michael - Microsoft Bot Framework

// Azahar Machwe - Deep Learning: An Introduction

Artificial Intelligence has come to the forefront in recent years with some of the biggest tech companies investing heavily in AI based products and services.
Deep Learning is one of the main reasons for this renewed interest.
This talk will provide an introduction to Deep Learning, provide use-cases and show how it is pushing the boundaries of what machines are capable of. It will also provide
information about popular Deep Learning libraries.

// Simon Michael - Microsoft Bot Framework and Cognitive Services

Developers writing bots all face the same problems: bots require basic I/O; they must have language and dialog skills; they must be performant, responsive and scalable; and they must connect to users, ideally in any language the user chooses. Multiply this by each conversational channel you want to target and you have a real challenge!

The Microsoft Bot Framework is a comprehensive offering to build and deploy high quality bots for your users to interact with using their favourite messaging applications. In this session you will learn about why bots are relevant, how to get started with the Microsoft Bot Framework, how to give your bot smart skills using Microsoft Cognitive Services such as natural language processing, and learn how to make a bot truly great!

// About Simon

Simon is a Senior Technical Evangelist within the Developer Experiences (DX) team.
He is currently technical lead on conversational UI covering Microsoft Bot Framework, Cortana and Microsoft Cognitive Services.
