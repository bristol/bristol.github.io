Go(ctober) - OpenAPI and Go 1.13

Hello Bristol Gophers 👋!

More talks, more fun, more food 🍕 and more drinks 🍻. You Gophers know the drill!

Speakers:

🗣 Benjamin King - Go 1.13

Go 1.13 is a big change: the introduction of GOPROXY & GOSUMDB, the introduction of wrapping errors and most crucially "This release improves performance of most uses of defer by 30%."; such a big change deserves a talk! So I will talk about them.

Benjamin King is a Software Engineer at Tumelo. Ben is not particularly interesting but Tumelo obviously is.

🗣 Mat Evans - Being lazy with OpenAPI and Go

Documenting an API is one of the more boring parts of the job, but what if you could write one spec and have documentation and all the boilerplate for an API service in Go written for you? A quick tour through The openAPI spec and some of the tooling available in Go.

Mat Evans has been messing around with Go since 2013, when he was bored at work and thought he'd learn something new. He's worked with Go in advertising, sports data, and is currently contracting at News UK where he's helping to build a paywall-as-service.
